## About the Project

This portfolio website serves as a digital resume and personal space where I can share my projects, skills, and professional experience. It was created to provide a comprehensive overview of my journey in the field of Computer Science. The website is hosted on GitHub Pages, allowing easy access for anyone interested in my work.

## Technologies Used

- **HTML** – For structuring content on the site.
- **CSS** – For styling and layout, including a modern, minimalist design.
- **Font Awesome** – For icons used throughout the site.
- **GitHub Pages** – For hosting and deploying the website.

## Getting Started

To set up this portfolio locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   cd portfolio-website
   ```

2. **Open the `index.html` file** in a browser to view the website locally.

## Hosting on GitHub Pages

1. Make sure your repository is public.
2. Go to **Settings** > **Pages** in your repository.
3. Under **Source**, select `Deploy from a branch`.
4. Choose the `main` branch (or `master` branch, depending on your setup), and save.
5. Your website will be available at `https://yourusername.github.io/portfolio-website`.
