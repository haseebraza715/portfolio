## Setup
To view locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/haseebraza715/portfolio.git
   cd portfolio
   ```
2. Open `index.html` in a browser.

To host on GitHub Pages:
1. Make the repository public.
2. Go to **Settings > Pages** and set the source to your branch (e.g., `main`).
3. Your site will be live at `https://haseebraza715.github.io/portfolio`.
